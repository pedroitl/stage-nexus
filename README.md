# stage-nexus
Stage Nexus é uma rede social voltada para artistas e profissionais do entretenimento ao vivo. O projeto apresenta páginas de Login, Feed, Perfil, Oportunidades e Inbox, utilizando HTML, CSS, JavaScript e React via CDN para componentes reutilizáveis. Desenvolvido como projeto acadêmico para a disciplina de Linguagens Interpretadas.
